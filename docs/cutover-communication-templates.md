## Template 4: Rollback Notification

### English

**Subject:** Email system migration reverted

Dear User,

Due to technical issues encountered during the migration, we have reverted to the previous email system.

**What happened:**
- The migration encountered unexpected technical issues
- To ensure service continuity, we have reverted to the previous system
- All your data remains intact on the original system

**What this means for you:**
- No action required
- Your email service continues as normal
- The original system is fully operational

**Next steps:**
- We are investigating the issues
- A new migration attempt will be scheduled once resolved
- You will be notified when a new migration date is set

We apologize for any inconvenience and appreciate your understanding.

Best regards,
IT Team

## Usage Guidelines

### When to Send Each Template

| Template | Timing | Recipients |
|----------|--------|------------|
| Pre-Cutover | 24 hours before | All affected users |
| In Progress | At cutover start | All affected users |
| Complete | Immediately after success | All affected users |
| Rollback | Immediately after rollback | All affected users |
| Support | 1-2 days after completion | All affected users |

### Customization Notes

- Replace bracketed placeholders `[like this]` with actual values
- Adjust timing based on your organization's maintenance windows
- Add company-specific contact information
- Include links to your internal support resources
- Consider adding branding/logo to templates

### Translation Quality

- These templates are written in clear, simple language
- Technical terms are kept minimal
- For organizations with non-native speakers, consider professional translation review
- Test translations with native speakers if possible
